# Evaluación U4 - GitHub Desktop y Markdown

## Proceso de Configuración

1. Abrí **GitHub Desktop** en mi computadora.
2. Hice clic en **"File" > "New Repository"**.
3. En el campo "Name" escribí: `Evaluacion-Markdown-MiNombreMiApellido`.
4. Seleccioné la carpeta local donde quería crear el repositorio.
5. Dejé marcada la opción **"Initialize this repository with a README"**.
6. Hice clic en **"Create Repository"**.
7. Una vez creado localmente, fui a **"Repository" > "Publish Repository"**.
8. Seleccioné **"Keep this code private"** (luego lo cambié a público desde la web).
9. Hice clic en **"Publish Repository"**.
10. Verifiqué en github.com que el repositorio estaba visible y público.
11. Edité la visibilidad desde la web de GitHub cambiando a **"Public"**.
